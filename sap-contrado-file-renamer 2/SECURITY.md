# Security Policy

Never commit real SAP documents, Contrado production reports, customer data, email addresses, credentials, tokens, internal URLs, company server paths, or confidential screenshots.

Use synthetic test values such as:

```text
Con No: 87654321
Reference: TEST-CONTRADO-001
```

Before production use, replace test paths with approved paths, confirm folder permissions, review overwrite behaviour and retention requirements, and test with controlled non-production data.
