# GitHub Setup

1. Sign in to GitHub and create a new repository named `sap-contrado-file-renamer`.
2. Make it **Private first** while checking the contents.
3. Do not initialise it with another README or license.
4. Extract this package.
5. On GitHub choose **Add file** → **Upload files**.
6. Upload the contents of the `sap-contrado-file-renamer` folder.
7. Commit with:

```text
Initial public release v1.3 with MIT license and documentation
```

8. Complete `docs/PUBLISHING_CHECKLIST.md`.
9. Change the repository to Public only after approval.

Suggested release:

```text
Tag: v1.3.0
Title: SAP–Contrado File Renamer v1.3
```
